# Full-Stack React Native & Express Integration

A robust demonstration of a full-stack mobile application architecture using **React Native (Expo)**, **Node.js (Express)**, and **MySQL**. This project showcases a seamless data flow from front-end entry forms to back-end persistent storage, featuring automated navigation and real-time data retrieval.

This is designed to demonstrate the core principles of modern web and mobile development. It bridges the gap between client-side user interaction and server-side logic, providing a blueprint for data-driven applications.

### Key Features
- **Dynamic Data Entry**: A clean, intuitive form for capturing user information.
- **RESTful API**: A Node.js backend handling secure POST and GET requests.
- **Persistent Storage**: Integration with MySQL for reliable data management.
- **Automated Routing**: Instant transition from submission to data visualization.
- **Responsive Display**: A dedicated results page that fetches and lists records directly from the live database.
